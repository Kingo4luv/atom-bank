### Atom Bank



## Description

A simple api to simulate bank transfer in nextjs

## Installation

1. clone repo
2. cd in atom-bank

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Api documentation for localhost

1. [Api Documentation](http://localhost:3000/api) web page.
2. [Api Documentation JSON](http://localhost:3000/api-json) Json Format.
