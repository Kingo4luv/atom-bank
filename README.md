### Atom Bank



## Description

A simple api to simulate bank transfer in nextjs

## Installation
1. clone repo
2. cd in atom-bank

3.```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
